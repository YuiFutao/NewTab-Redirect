# new tab colors-of-the-day

## Google Chrome Extension
### 概要
曜日を連想させる写真を新しいタブに表示します

### 詳細説明
新しいタブを開いたときに、日付・時刻を表示します。
ミニマルなデザインで、美しい背景画像が曜日を連想させます。
